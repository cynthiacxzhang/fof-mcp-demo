# Flow-of-Funds MCP Server
A demo MCP server connecting an AI agent to banking transaction data — schema introspection, SQL execution, and pre-built analytics tools.

Deployed: [fof-mcp-demo.up.railway.app](https://fof-mcp-demo.up.railway.app/)

## Stack
- [MCP](https://modelcontextprotocol.io) via `fastmcp`
- DuckDB (synthetic data stand-in)
- Python 3.11+, uv
- Railway + Anthropic
